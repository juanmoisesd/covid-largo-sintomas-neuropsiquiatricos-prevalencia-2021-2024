## Descripción

Estadísticas globales y por país sobre los **síntomas neurológicos y psiquiátricos del COVID Largo** (PASC), procedentes de revisiones sistemáticas y estudios de cohortes (2021-2024).

## Prevalencia de síntomas clave

| Síntoma | Rango global | Ejemplo por país |
|---|---|---|
| Fatiga | **74%** | Bangladesh: 82,9% → 16,1% (semana 31) |
| Niebla mental | **32-60%** | Colombia: 60% a los 2-3 años |
| Trastornos del sueño | **41-64%** | EE.UU. no hospitalizados: 64% |
| Malestar postesfuerzo | **58,7%** | Cohorte FIOCRUZ Brasil |
| Ansiedad | **23-44%** | Colombia: 44% |
| Depresión | **12-30%** | Criterios diagnósticos |
| Deterioro cognitivo | **17,5-33%** | OCDE: 16-33% |
| Dolor muscular | **42%** | Cohortes globales |
| Entumecimiento | **41%** | Cohortes globales |

**1/6 a 1/3** de infectados en países OCDE presentan síntomas cognitivos persistentes.
